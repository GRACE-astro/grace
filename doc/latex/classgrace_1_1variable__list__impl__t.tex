\doxysubsection{Detailed Description}
Implementation of the variable list type. 

This class is wrapped with a {\ttfamily singleton\+\_\+holder} and is used as a utility to collect all variables and coordinates and ensure they have the appropriate lifetime during GRACE\textquotesingle{}s execution.
