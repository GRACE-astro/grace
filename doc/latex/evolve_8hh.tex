\begin{DoxyCompactList}\small\item\em Perform a timestep.

This function advances all variables in the state array by a full timestep. The timestep size is controlled by the function find\+\_\+stable\+\_\+timestep. The kind of timestepper used is controlled by the parameter evolution\+::time\+\_\+stepper. Coming out of this routine all the variables in the state array are in a valid state (at all gridpoints) and at time $t+dt$. Auxiliaries are not filled by this function and neither is the scratched space, both of which are left in an invalid state. This function assumes that the state is in a valid state as input, for all gridpoints including ghostzones. It also assumes that auxiliaries are filled at all gridpoints and up to date w.\+r.\+t. the evolution time. \end{DoxyCompactList}\item
