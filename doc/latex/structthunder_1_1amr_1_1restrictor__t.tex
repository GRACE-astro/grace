\[ \frac{\sum_{\{i,j,k\} \in I^{(i_0,j_0,k_0)}_{\rm c} } U^{l+1}_{i,j,k}\,V^{l+1}_{i,j,k}} {\sum_{\{i,j,k\} \in I^{(i_0,j_0,k_0)}_{\rm c} } V^{l+1}_{i,j,k} } \] Where $ I^{(i_0,j_0,k_0)}_{\rm c} $ contains the indices of children cells of the coarse cell $(i_0,j_0,k_0)$.
