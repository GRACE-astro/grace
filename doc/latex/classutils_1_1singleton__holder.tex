A \mbox{\hyperlink{classutils_1_1singleton__holder}{singleton\+\_\+holder}} manages the lifetime of a unique object of type {\ttfamily T}. It ensures that at most one instance of {\ttfamily T} exists at any given time and that the singleton object lives as long as required by the application. It also provides a static method to access the reference to the singleton itself.
